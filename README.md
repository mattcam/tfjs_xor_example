# tfjs_xor_example

> TensorFlow JavaScript XOR example

Motivation for creating TensorFlow Javascript XOR example was to learn and explore how to solve a basic machine learning problem using TensorFlow JavaScript.

## Prerequisites

Node

## Usage

```
npm install
npm run start
```
